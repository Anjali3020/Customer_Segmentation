# Customer Segmentation Using Python

## Project Description

This project classifies customers into different segments based on their purchase amount.

## Technologies Used

- Python
- Pandas
- CSV File Handling

## Files

1. customer_segmentation.py
2. Customer.csv
3. segmented_customers.csv

## Customer Segments

- Premium : Purchase Amount >= 20000
- Gold : Purchase Amount >= 15000
- Silver : Purchase Amount >= 8000
- Regular : Purchase Amount < 8000

## How to Run

1. Install pandas

pip install pandas

2. Run the program

python customer_segmentation.py
